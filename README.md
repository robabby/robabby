## Hey there!

Thanks for taking a look at the code for my website at [robabby.com](http://robabby.com).

---

This website is meant to serve as my personal website with a blog.  The client side is built with Ember.js.  The server is built with Node.js API running queries against a Mongolab Database.  I am currently using [Express.js](http://expressjs.com/) for the server but plan on refactoring to [Hapi.js](http://hapijs.com/) when I get the client more fleshed out.

Pull requests welcome!
